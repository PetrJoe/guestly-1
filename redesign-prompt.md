# 🎯 GUESTLY Platform Redesign Brief

**Objective:** Create a modern, youthful, premium event commerce platform with strong visual identity, immersive user experience, and full dark mode support.

---

## 1. Project Vision

Redesign GUESTLY into a vibrant, emotionally engaging, and premium event ticketing and event management platform.

The platform must feel:

* Modern
* Energetic
* Youthful but professional
* Elegant yet bold
* Technologically advanced
* Trustworthy and secure

GUESTLY should visually communicate:

* Movement
* Experiences
* Community
* Creativity
* Structure
* Reliability

This is not just a ticketing website.
It is an event commerce ecosystem.

---

## 2. Brand Color System

Use the following palette as the core identity:

Primary Blue: `#4392F1`
Deep Navy: `#0D1821`
Pure White: `#FFFFFF`
Accent Red: `#DF2935`
Emerald Green: `#248232`

### Color Strategy

* Blue represents trust and technology.
* Navy represents depth and structure.
* White creates space and clarity.
* Red adds energy and urgency.
* Green represents growth and financial flow.

### Requirements

* Fully optimized light mode and dark mode.
* Dark mode should not simply invert colors. It must feel intentionally designed.
* Maintain accessible contrast ratios.
* Use accent colors strategically for CTA, alerts, and financial indicators.

---

## 3. Design Language Direction

### Visual Style

* Clean grid based layouts
* Large typography hierarchy
* Soft shadows and elevation layering
* Rounded corner components
* Glass morphism or subtle blur effects where appropriate
* Smooth micro interactions
* Fluid hover states and motion transitions
* Minimal but expressive iconography

The UI should feel closer to Stripe, Linear, Notion, Framer, or Airbnb dashboards in polish.

Avoid clutter.
Avoid overuse of gradients.
Avoid chaotic color mixing.

---

## 4. Core Platform Areas To Redesign

### A. Public Website

* Hero section with dynamic event visuals
* Smooth animated transitions
* Modern event cards
* Intelligent filtering and search
* Personalized recommendations
* Strong call to action hierarchy
* Featured and sponsored event sections
* Social proof blocks

The homepage must feel alive.

---

### B. Organizer Dashboard

Design a world class dashboard experience.

Must include:

* Revenue analytics
* Ticket sales graphs
* Real time sales feed
* Settlement tracker
* Promo code performance
* Event status overview
* Attendee insights
* Campaign tracking
* Quick action panels

Design requirements:

* Modular widget based layout
* Customizable dashboard components
* Smooth transitions between data views
* Beautiful data visualization design
* Clean chart styling
* Smart empty states

The dashboard should feel powerful but simple.

---

### C. Admin Control Panel

* Global platform metrics
* Event performance breakdown
* Commission tracking
* Settlement control
* User management
* Dispute and refund management
* Sponsorship placements control

The admin area must feel enterprise grade and secure.

---

### D. Ticket Purchase Flow

Redesign checkout for:

* Speed
* Simplicity
* Trust

Include:

* Clear pricing breakdown
* Smooth multi step process
* Mobile optimized layout
* Animated confirmation experience
* Digital wallet ready design

The checkout must feel safe and seamless.

---

## 5. Dark Mode Requirements

Dark mode must:

* Use Deep Navy as foundation
* Adjust surfaces with layered tonal variations
* Maintain readable typography
* Soften bright accents
* Include glow effects for CTA buttons
* Preserve brand consistency

Dark mode should feel premium, not gloomy.

---

## 6. Component Design System

Create a scalable design system including:

* Buttons (primary, secondary, ghost, danger)
* Input fields
* Dropdowns
* Tabs
* Cards
* Alerts
* Toast notifications
* Charts
* Tables
* Badges
* Status indicators
* Modals
* Empty states
* Loaders and skeleton states

Everything must be reusable and scalable.

Design must support future white label customization.

---

## 7. Mobile First Optimization

GUESTLY must feel native on mobile.

* Responsive grid
* Thumb friendly interactions
* Clean typography scaling
* Collapsible dashboards
* Smooth bottom navigation where necessary

Mobile experience is not secondary.
It is primary.

---

## 8. Motion and Interaction Design

Include:

* Micro animations
* Button ripple effects
* Smooth page transitions
* Animated data loading states
* Smart hover elevation
* Subtle motion feedback

Motion should feel intentional and premium.

---

## 9. Typography Direction

* Modern sans serif
* Strong heading contrast
* Clear data readability
* Comfortable paragraph spacing

Typography must communicate authority and clarity.

---

## 10. Emotional Outcome

After redesign, users should feel:

* This platform is serious.
* This platform is modern.
* This platform can handle large scale events.
* This platform understands creators.
* This platform feels global.
